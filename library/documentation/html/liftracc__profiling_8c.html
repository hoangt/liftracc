Detailed Description

C file of the profiling component.

Author:
Manuel Niekamp <niekma@upb.de>
Version:
0.1

This file implements the profiling component. Profiling is used to measure timings. Functionality to start and stop a time measurement is provided. It is possible to save the collected data to a file.
