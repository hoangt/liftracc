Detailed Description

Header file of the profiling component.

Author:
Manuel Niekamp <niekma@upb.de>
Version:
0.1

This file defines the interface to the profiling component. Profiling is used to measure timings. Functionality to start and stop a time measurement is provided. It is possible to save the collected data to a file.
