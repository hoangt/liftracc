libliftracc Documentation

0.1

This is the source code documentation of the liftracc library. Information on how to set up and use this library is presented.

Prerequirements

  • C/C++/Fortran compiler
  • cmake+make
  • netlib.org blas+cblas library
  • python (for the generator scripts)

Optional

  • additional BLAS libraries (Goto2, ATLAS, MKL, ...)
  • hardware accelerated versions of BLAS (cuda, clearspeed, ...)
