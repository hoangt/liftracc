Detailed Description

C file of the selector module.

Author:
Manuel Niekamp <niekma@upb.de>
Version:
0.1
Date:
10/2009-03/2010

This file implements the interface to the selector component. The selector is used inside the wrapper library to select the best function out of a function pool. The functions selected match the calling one from the wrapper library.
